# Ahmad Niknam Portfolio Website

A professional personal portfolio website for Ahmad Niknam that combines IT portfolio content with portfolio website design services.

## Live Website

https://ahmadniknam.com

## Key Features

- Professional hero section
- About / professional background
- Skills and technical areas
- IT projects
- Portfolio Website Design Services section
- Demo portfolio website concepts
- Simple portfolio website process
- Contact section with LinkedIn and email
- Responsive design for desktop, tablet, and mobile

## Technology Stack

- Next.js
- React
- TypeScript
- Tailwind CSS
- Vercel

## Portfolio Website Services

The website includes a service section for helping IT professionals, students, researchers, job seekers, newcomers, project managers, consultants, and freelancers create clean, professional portfolio websites.

## Demo Portfolio Concepts

- IT Professional Portfolio
- Graduate Student / Researcher Portfolio
- Project Manager Portfolio
- Newcomer Career Portfolio

These are sample website concepts, not completed client projects.

## Local Development

Install dependencies:

```bash
npm install
```

Start the development server:

```bash
npm run dev
```

Run lint checks:

```bash
npm run lint
```

Build the production version:

```bash
npm run build
```

## Deployment

The website is deployed on Vercel and connected to the main GitHub branch. Pushing changes to `main` triggers a production deployment.
