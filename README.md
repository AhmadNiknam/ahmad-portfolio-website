# Ahmad Niknam Portfolio

A modern personal portfolio website for Ahmad Niknam, an IT Infrastructure and Cloud professional based in Regina, Saskatchewan, Canada.

## Stack

- Next.js App Router
- TypeScript
- Tailwind CSS
- Responsive component-based layout

## Getting Started

Install dependencies and run the development server:

```bash
npm install
npm run dev
```

Open `http://localhost:3000` to view the site.
